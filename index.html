Welcome to Signal R Angular Chat.

This is simple feature rich chat web-based chat application, developed using Signal R and Angular Js.

Main features of this application are :

  • Shows list of logged-in users.
  • Instant notification if any user come online or goes offline
  • Option to update status[Online, Busy, Away], status is updated instantly to all other users.
  • User in chat's activity [like Skype's 'is typing', 'is deleting message'.
